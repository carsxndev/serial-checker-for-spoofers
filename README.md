# Project Overview

This project is a **simple Windows Batch (.bat) script** created for **educational and demonstration purposes**. It shows how basic command-line operations can be automated using Batch scripting on Windows systems.

---

## Language

* **Windows Batch (.bat)**
* Runs on **Windows Command Prompt (CMD)**
* No external dependencies required

---

## Purpose

The main goal of this project is to:

* Demonstrate basic Batch scripting logic
* Automate simple command-line tasks
* Provide a beginner-friendly example

---

## Usage

1. Download or clone the repository
2. Double-click the `.bat` file or run it via CMD

```bat
Serials.bat
```

---

## Notes

* This project is **not intended for production use**
* It is meant for learning and testing only
